# Music resources

Cell tables extracted from **BANA 2015 *Music Braille Code***
(`Music_Braille_Code_2015.pdf`, project root).

Every file is `status: "unchecked"` until a human reviewer
spot-checks the cells against the PDF table.

Generated by `scripts/extract_music_tables.py`. Do not edit by
hand -- change the script's `TABLE_*` literals and re-run.

## Notes on cell references

- `c_<dots>` follows `resources/cells.json` (e.g. `c_13456` =
  dots 1+3+4+5+6 = â ˝). All references must resolve there.
- `c_blank` is a sentinel for the empty cell (no dots). It is
  emitted whenever an entry's ASCII token contains an internal
  space (BANA prints two cell groups separated by a literal
  space character). The backend treats `c_blank` as 'emit zero
  dots'.
